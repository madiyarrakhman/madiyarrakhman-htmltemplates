---
description: Запуск процесса тестирования проекта согласно чек-листу
---

Для проведения тестирования выполните следующие шаги:

1. Открой и изучи актуальный чек-лист в файле `TESTING_CHECKLIST.md`.
2. Проверь статус API сервера и фронтенда (запущены ли процессы).
3. Последовательно пройди по разделам:
    - **Frontend**: визуальная проверка и формы.
    - **Admin**: функционал входа и управления.
    - **API**: проверка эндпоинтов через curl или внутренние тесты.
4. Если были добавлены новые функции, обнови `TESTING_CHECKLIST.md` перед началом.
5. Зафиксируй результаты тестирования в логах или ответе пользователю.
